# Portrait Action Asset Notes

These PNG files are local photo-style action thumbnails for the portrait action selector.

- Purpose: show the available portrait action presets as real raster image cards in the UI.
- Source workflow: locally generated deterministic drawing assets for this project; no external photos, logos, watermarks, brand marks, or sourced catalog images are included.
- Content standard: generic adult portrait poses only, no identifiable person, no sensitive trait implication, and no sexualized framing.
- Asset names: `action-standing.png`, `action-walking.png`, `action-seated.png`, `action-leaning.png`, `action-looking-back.png`, `action-adjusting-sleeve.png`, `action-holding-prop.png`, and `action-turning.png`.
