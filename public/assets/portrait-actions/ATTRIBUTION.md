# Portrait Action Asset Notes

These PNG files are local photorealistic action thumbnails for the portrait action selector.

- Purpose: show the available portrait action presets as real raster image cards in the UI.
- Source workflow: locally generated with the built-in image generation workflow as a neutral studio pose-reference sheet, then cropped into individual app assets; no external photos, logos, watermarks, brand marks, or sourced catalog images are included.
- Content standard: generic adult portrait poses only, non-identifiable face treatment, no sensitive trait implication, and no sexualized framing.
- Asset names: `action-standing.png`, `action-walking.png`, `action-seated.png`, `action-leaning.png`, `action-looking-back.png`, `action-adjusting-sleeve.png`, `action-holding-prop.png`, and `action-turning.png`.
