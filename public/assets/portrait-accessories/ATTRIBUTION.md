# Portrait Accessory Asset Notes

These files are local white-background product-reference assets for the built-in portrait accessory library.

- Composition standard: one product per image, front-facing or near-front product catalog view, centered on a clean white background.
- Content standard: no people, no live model scenes, no room background, no brand marks, no watermarks, no labels, and no cluttered crop.
- Source workflow: generated as photorealistic retail product photography sheets, then cropped into individual app assets.
- Classic style color variants: locally generated white-background product thumbnails for upper, bottom, and shoe classics; each variant is a standalone PNG preview with no sourced product photo, brand mark, watermark, or catalog image.
- COS category additions: local generated cosplay character reference assets using generic anime-inspired and fantasy character archetypes; no official character, franchise, actor, celebrity, logo, catalog image, or sourced fan art is included.
